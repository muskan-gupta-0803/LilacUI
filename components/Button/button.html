Button

Primary Button

Primary button is the main button design in project.

Secondary Button

Secondary button is the button that is mainly used alonside a primary button.

Button with a link

Buttons with link contains link to some external/internal pages.

Button with Icon

Buttons with icons contain related icon along with text.
