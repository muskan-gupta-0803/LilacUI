IMAGE

To start using the image components in your project, use the code snipppets below.

Round-Image

A round image is the one that is circular in shape. Use class "img-round" for this component.

Responsive-Image

A Responsive image is the one that adjusts its height and width according to the screen size. Use class "img-responsive" for this component.
